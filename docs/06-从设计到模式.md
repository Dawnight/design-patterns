# 7. 如何讲解设计模式
+ 介绍和举例(生活中易理解的示例)
+ 画UML类图写demo代码
+ 结合经典应用场景，讲解该设计模式如何被使用

# 8. 如何学习设计模式
+ 明白每个设计模式的道理和用意
+ 通过经典应用体会它的真正使用场景
+ 自己编码的时候多思考，尽量模仿
