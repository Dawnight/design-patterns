# 1. UML类图
+ Unified Modeling Language(统一建模语言)
+ 类图，UML包括很多种图
+ 关系，主要讲解泛化和关联
+ 工具: MS Office visio， https://www.processon.com

# 2. 介绍
+ 类名
	+ `+` public 属性名A: 类型
	+ `#` protected 属性名B: 类型
	+ `-` private 属性名C: 类型
	+ `+` public 方法名A(参数1，参数2): 返回值类型
	+ `#` protected 方法名B(参数1，参数2): 返回值类型
	+ `-` private 方法名C(参数1): 返回值类型

# 3 关系
+ 泛化: 表示继承
+ 关联: 表示引用
